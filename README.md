# FitnessApp Bot | Telegram Mini App
FitnessApp — это многофункциональный Telegram-бот для тех, кто следит за своим телом. Он объединяет в себе два инструмента: точный счетчик КБЖУ и дневник тренировок для отслеживания прогрессии нагрузок.
# Основные возможности
 Питание и КБЖУ

    База продуктов: Быстрый поиск и добавление продуктов.

    Расчет нутриентов: Автоматический подсчет белков, жиров и углеводов.

    Дневной лимит: Визуализация прогресса потребления калорий за день.
    
 Тренировочный дневник

    Прогрессия нагрузок: История подходов и весов для каждого упражнения.

    Аналитика: Графики роста силовых показателей.
